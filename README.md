## blind-printing
# Приложение-тренажер слепой печати

# Сделано по аналогии с https://www.ratatype.com/ru/typing-test/test/

# Запуск проекта локально

### Установка node 18.16.0
- nvm install 18.16.0
- nvm use 18.16.0

### Установка зависимостей через yarn
- yarn

### Запуск приложения 
- yarn dev
